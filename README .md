# 🌤️ Weather Data Analysis with Python

A beginner-friendly data analysis project that explores daily weather data using **Python, Pandas, and Matplotlib** to identify trends and patterns in temperature and humidity.

---

## 📌 Project Overview

| Detail | Info |
|--------|------|
| **Dataset** | Daily weather records (temperature, humidity) |
| **Type** | Data Analysis & Visualization |
| **Tech stack** | Python, Pandas, Matplotlib |
| **Level** | Beginner |

---

## 📁 Files

| File | Description |
|------|-------------|
| `Project1.py` | Main Python script — analysis + visualizations |
| `weather_data.csv` | Dataset with daily weather records |
| `requirements.txt` | All dependencies |

---

## 🚀 How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Run the analysis
python Project1.py
```

---

## 📊 What the Project Does

- Loads and cleans the weather dataset using **Pandas**
- Calculates key statistics — average temperature, average humidity
- Finds days above a temperature threshold (>25°C)
- Finds days below a humidity threshold (<55%)
- Creates 3 visualizations:
  - 📈 Line chart — Temperature over time
  - 📊 Histogram — Humidity distribution
  - 📉 Dual-axis chart — Temperature & Humidity together

---

## 📈 Sample Visualizations

### Temperature Over Time
Line chart showing how temperature changes day by day.

### Humidity Distribution
Histogram showing how humidity is spread across all recorded days.

### Temperature & Humidity — Dual Axis
Combined chart showing the relationship between temperature and humidity over time.
